# Cyber-Security_Task_02_RadhikaSonar

### CIA Triad & Risk Assessment

## About This Task
This task helps students understand the fundamental principles of cybersecurity and how organizations identify, analyze, and manage security risks. It focuses on the CIA Triad, cyber threats, risk assessment, and security recommendations.
The CIA Triad forms the foundation of cybersecurity and helps protect information systems from various cyber attacks.

## Objectives
* Understand the CIA Triad concepts.
* Learn the importance of information security.
* Study real-world cyber attacks.
* Perform risk assessment activities.
* Identify common cyber threats.
* Suggest security improvements for organizations.

## Task Contents
This task focuses on the fundamental concepts of cybersecurity, including the CIA Triad, risk assessment, cyber threats, and security practices. It helps in understanding how organizations protect sensitive information, maintain data accuracy, and ensure system availability. The task also includes studying a real-world cyber attack, identifying risks to important assets, analyzing common security threats, and suggesting effective security measures to improve cybersecurity awareness and reduce potential risks.

## Key Learnings
* The CIA Triad is the foundation of cybersecurity.
* Risk assessment helps identify security issues.
* Cyber attacks can affect organizations severely.
* Security awareness reduces cyber risks.
* Preventive measures improve overall security.

## Conclusion
Cybersecurity is essential for protecting information, systems, and users. This task provided knowledge about the CIA Triad, risk management, cyber threats, and security best practices. By implementing proper security measures, organizations can reduce risks and improve their cybersecurity posture.
